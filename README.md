# 3D-Visualization-of-the-GRB-SN-connection-1-

Here we present the 3D interactive plot of the Type-II GRBs and the SNe connection. (A sample of the 3D Plots can be found in the main branch of the repository)

The code for the 3D plot and the interactive 3D plot is available in this repository in the form of Python notebook (IPYNB file) along with the datsets required to generate the plots in a ZIP file. 

Along with the interactive 3D Plot, the code also creates animation videos of the 3D plot for better visualisation. 

As mentioned in our research paper we have created some of the attached datasets from different datasets and other by predicting using the 3D KDE model. 
For more reference please visit our research article on the GRB-SN connection.
